# نظام اختيار هدف الإعلان (Popup Target System)

## نظرة عامة 📋

تم تطوير نظام جديد لإعلانات المنبثقة يسمح لك باختيار الهدف (صفحة أو تصنيف أو منتج) بدلاً من إدخال رابط خارجي فقط.

## الميزات الجديدة ✨

### 1. **اختيار من ثلاثة أنواع من الأهداف:**

#### أ) صفحات الموقع (Pages)
- الصفحة الرئيسية (Home)
- جميع المنتجات (All Products)
- من نحن (About)
- تواصل معنا (Contact)

#### ب) التصنيفات (Categories)
- العناية بالشعر (Hair Care)
- العناية بالبشرة (Skincare)
- مكياج (Makeup)
- عطور (Perfumes)
- سيرومات (Serums)
- ماسكات (Masks)
- زيوت (Oils)

#### ج) المنتجات (Products)
- اختر أي منتج من متجرك
- البحث السريع للمنتجات
- عرض صورة المنتج في قائمة الاختيار

### 2. **سهولة الاستخدام:**
```
1. انتقل إلى إدارة الإعلانات
2. أنشئ إعلان جديد أو عدّل إعلان موجود
3. في قسم "اختر الهدف":
   - اختر نوع الهدف (صفحة/تصنيف/منتج)
   - ابحث واختر الهدف المطلوب
   - سيظهر الهدف المختار في تأكيد أسفل الخيارات
4. احفظ الإعلان
```

## تحقيق الفوائد 🎯

### تحسين تجربة المستخدم:
- **روابط داخلية أسرع** - لا تحتاج إلى الانتظار لتحميل صفحة خارجية
- **ملاحة سلسة** - البقاء داخل التطبيق بسلاسة
- **أيقونة أفضل** - أيقونة سهم بدلاً من أيقونة رابط خارجي للروابط الداخلية

### سهولة الإدارة:
- **واجهة مرئية** - اختر بصرياً بدلاً من كتابة روابط
- **بحث متكامل** - ابحث عن المنتجات والتصنيفات بسهولة
- **خيارات محدودة** - تجنب الأخطاء في الروابط

## خطوات التنفيذ 🚀

### الخطوة 1: تحديث قاعدة البيانات
1. افتح [Supabase Dashboard](https://app.supabase.com)
2. اختر مشروعك
3. اذهب إلى **SQL Editor**
4. افتح ملف `POPUP_TARGET_SYSTEM_MIGRATION.sql`
5. انسخ جميع الأوامر
6. الصقها في SQL Editor
7. انقر على **Run** أو اضغط `Cmd+Enter` / `Ctrl+Enter`

✅ تم! الآن لديك الأعمدة الجديدة في قاعدة البيانات.

### الخطوة 2: استخدام النظام الجديد
1. انتقل إلى **صفحة إدارة الإعلانات**
2. **أنشئ إعلان جديد** أو **عدّل إعلان موجود**
3. ستلاحظ **قسم جديد** بعنوان "اختر الهدف"
4. اتبع الخطوات أعلاه لاختيار هدفك

## الملاحظات المهمة ⚠️

### التوافقية العكسية
- **الروابط القديمة لا تزال تعمل** ✅
- إذا كان لديك إعلان قديم يستخدم `link_url`، سيستمر في العمل
- يمكنك الخلط بين الإعلانات - بعضها يستخدم النظام القديم والبعض الآخر الجديد

### أيقونات الأزرار
- **روابط داخلية** → أيقونة سهم ➡️
- **روابط خارجية** → أيقونة رابط خارجي 🔗

### التنقل السلس
- عند النقر على زر إعلان داخلي:
  1. يُغلق الإعلان تلقائياً
  2. يتم الانتقال إلى الصفحة/المنتج/التصنيف
  3. لا توجد نوافذ منفصلة

## الاختبار 🧪

### لاختبار النظام الجديد:
1. افتح الصفحة الرئيسية
2. انتظر ظهور الإعلان (أو اختبره من صفحة الإدارة بزر "معاينة")
3. اضغط على الزر
4. يجب أن ينقلك إلى الهدف المختار

### استكشاف الأخطاء:
```
إذا لم ينجح الانتقال:
1. افتح Developer Console (F12)
2. ابحث عن الأخطاء في قسم Console
3. تحقق من أن:
   - الإعلان يحتوي على target_type و target_id
   - أو يحتوي على link_url (النظام القديم)
```

## أمثلة 📚

### مثال 1: إعلان للصفحة الرئيسية
```
نوع الهدف: صفحة
الهدف: الصفحة الرئيسية
النص: "عودة للصفحة الرئيسية"
النتيجة: رابط داخلي → /
```

### مثال 2: إعلان لتصنيف معين
```
نوع الهدف: تصنيف
الهدف: العناية بالشعر
النص: "اكتشف منتجات الشعر"
النتيجة: رابط داخلي → /products?category=hair_care
```

### مثال 3: إعلان لمنتج محدد
```
نوع الهدف: منتج
الهدف: منتج معين (يظهر اسمه وصورته)
النص: "تسوق الآن"
النتيجة: رابط داخلي → /products/product-slug
```

## الدعم والمساعدة 💬

إذا واجهت أي مشاكل:
1. تحقق من أن الأعمدة الجديدة تمت إضافتها في Supabase
2. أعد تحميل الصفحة (Ctrl+R أو Cmd+R)
3. جرب مسح ذاكرة التخزين المؤقت (Ctrl+Shift+Delete)
4. تحقق من console من أجل الأخطاء

## الملخص 🎉

- ✅ **نظام أسهل** لإنشاء إعلانات موجهة
- ✅ **تجربة مستخدم أفضل** مع ملاحة سلسة
- ✅ **توافقية كاملة** مع الإعلانات القديمة
- ✅ **واجهة مرئية وسهلة** للاختيار
