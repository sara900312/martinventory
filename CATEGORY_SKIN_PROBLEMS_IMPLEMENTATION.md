# تعديل عرض مشاكل البشرة/الشعر أسفل المنتجات

## الملفات المعدلة

### 1. `src/hooks/useCategorySkinProblems.js` (ملف جديد)
- **الوظيفة**: Hook لجلب مشاكل الجلد بناءً على فئة المنتج
- **المدخلات**: `productCategory` (مثل 'hair_care' أو 'skincare')
- **المخرجات**: Array من مشاكل الجلد المتعلقة بالفئة

**المقابلة (Mapping):**
- `'hair'` → `'العناية بالشعر'`
- `'hair_care'` → `'العناية بالشعر'`
- `'skincare'` → `'العناية بالبشرة'`
- `'skin_care'` → `'العناية بالبشرة'`

### 2. `src/components/ProductCard.jsx` (معدل)
- **تم الإضافة**: استيراد الـ hook `useCategorySkinProblems`
- **تم الإضافة**: استدعاء الـ hook مع `product.category`
- **تم الإضافة**: عنصر JSX جديد لعرض مشاكل الجلد المرتبطة

## مثال التدفق

### منتج في فئة Hair:
```
product.category = 'hair_care'
  ↓
useCategorySkinProblems('hair_care')
  ↓
getCategoryMapping('hair_care') → 'العناية بالشعر'
  ↓
SELECT * FROM skin_problems WHERE tool_title = 'العناية بالشعر'
  ↓
النتيجة:
- dandruff (قشرة الرأس)
- dry_hair (جفاف الشعر)
- hair_loss (تساقط الشعر)
- oily_hair (الشعر الدهني)
- split_ends (تقصف الشعر)
```

### منتج في فئة Skincare:
```
product.category = 'skincare'
  ↓
useCategorySkinProblems('skincare')
  ↓
getCategoryMapping('skincare') → 'العناية بالبشرة'
  ↓
SELECT * FROM skin_problems WHERE tool_title = 'العناية بالبشرة'
  ↓
النتيجة:
- acne (حب الشباب)
- dark_spots (البقع الداكنة)
- dryness (جفاف البشرة)
- oily_skin (البشرة الدهنية)
- sensitivity (البشرة الحساسة)
- wrinkles (التجاعيد)
```

## النقاط الرئيسية

✅ **منتجات الشعر** تعرض مشاكل الشعر فقط  
✅ **منتجات البشرة** تعرض مشاكل البشرة فقط  
✅ **لا خلط** بين فئات المشاكل المختلفة  
✅ **عرض أول 3 مشاكل** مع عداد للبقية إن وجدت  
✅ **استخدام Emoji** لتمييز المشاكل بصرياً  

## متطلبات قاعدة البيانات

تأكد من أن جدول `skin_problems` يحتوي على:
- ✅ العمود `tool_title` (موجود بعد migration)
- ✅ البيانات الصحيحة مع `tool_title = 'العناية بالشعر'` أو `'العناية بالبشرة'`
- ✅ الحقول: `id`, `name_ar`, `name_en`, `emoji`, `description`, `tool_title`

## الاختبار
