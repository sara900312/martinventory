# 🚀 دليل البدء السريع - نظام اختيار هدف الإعلان

## المتطلبات الأساسية
- Supabase مفعل وقيد التشغيل
- صفحة إدارة الإعلانات في المتجر

## الخطوات الثلاث السريعة

### ✅ الخطوة 1: تحديث قاعدة البيانات (5 دقائق)

1. افتح [Supabase Dashboard](https://app.supabase.com)
2. في **SQL Editor** الصق هذا الكود:

```sql
ALTER TABLE public.popup_hero
ADD COLUMN IF NOT EXISTS target_type VARCHAR(50) DEFAULT NULL,
ADD COLUMN IF NOT EXISTS target_id VARCHAR(255) DEFAULT NULL;
```

3. اضغط **Run** ✅

### ✅ الخطوة 2: أعد تحميل التطبيق

اضغط `F5` أو `Ctrl+R` لتحديث المتصفح

### ✅ الخطوة 3: ابدأ الاستخدام

اذهب إلى إدارة الإعلانات وستجد القسم الجديد "اختر الهدف"

---

## 📚 الاستخدام الأساسي

### إنشاء إعلان يوجه إلى الصفحة الرئيسية

```
1. انقر "إنشاء إعلان"
2. أضف العنوان والوصف
3. في "اختر الهدف":
   - اختر "صفحة"
   - اختر "الصفحة الرئيسية"
4. أضف نص الزر (مثلاً: "العودة للرئيسية")
5. احفظ الإعلان
```

### إنشاء إعلان يوجه إلى تصنيف

```
1. اتبع نفس الخطوات أعلاه
2. لكن اختر:
   - نوع الهدف: "تصنيف"
   - الهدف: "العناية بالشعر" (مثلاً)
3. احفظ
```

### إنشاء إعلان يوجه إلى منتج محدد

```
1. اتبع نفس الخطوات
2. اختر:
   - نوع الهدف: "منتج"
   - ابحث عن المنتج بالاسم
   - اختره من النتائج
3. احفظ
```

### إنشاء إعلان برابط خارجي

```
1. اتبع نفس الخطوات
2. اختر:
   - نوع الهدف: "رابط خارجي"
   - أدخل الرابط: https://example.com
3. احفظ
```

## 🎯 الخيارات المتاحة

### الصفحات الداخلية:
- ✅ الصفحة الرئيسية
- ✅ جميع المنتجات
- ✅ من نحن
- ✅ تواصل معنا

### التصنيفات:
- ✅ العناية بالشعر
- ✅ العناية بالبشرة
- ✅ مكياج
- ✅ عطور
- ✅ سيرومات
- ✅ ماسكات
- ✅ زيوت

### المنتجات:
- ✅ أي منتج من متجرك
- ✅ بحث سريع
- ✅ عرض صورة المنتج

### الروابط الخارجية:
- ✅ أي رابط https
- ✅ تفتح في نافذة جديدة

## 🔍 الاختبار

1. اذهب للصفحة الرئيسية
2. انتظر ظهور الإعلان (أو استخدم "معاينة")
3. اضغط الزر
4. تحقق من الانتقال إلى الهدف

---

## ⚡ نصائح سريعة

| المشكلة | الحل |
|--------|------|
| لا يظهر الإعلان | تأكد من `is_active = true` والتاريخ صحيح |
| لا تعمل الملاحة | أعد تحميل الصفحة (Ctrl+R) |
| لا توجد منتجات | تأكد من أن قاعدة البيانات بها منتجات |
| خطأ في الحفظ | افتح Console (F12) وابحث عن الأخطاء |
