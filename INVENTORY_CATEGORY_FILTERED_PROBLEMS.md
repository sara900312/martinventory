# تصحيح عرض مشاكل الجلد/الشعر في صفحة المخزن

## المشكلة السابقة
كانت صفحة إدارة المخزن (`InventoryPage`) تعرض **جميع** مشاكل الجلد والشعر معاً بغض النظر عن فئة المنتج المختارة.

## الحل المطبق

### الملف المعدل
**`src/components/ProductSkinRoutineSettings.jsx`**

### التغييرات
1. **إضافة دالة مقابلة (Mapping)**:
   ```javascript
   const getCategoryToolTitle = (cat) => {
     const mapping = {
       'hair': 'العناية بالشعر',
       'hair_care': 'العناية بالشعر',
       'skincare': 'العناية بالبشرة',
       'skin_care': 'العناية بالبشرة',
     };
     return mapping[cat?.toLowerCase()] || null;
   };
   ```

2. **تعديل الـ `useEffect` لتصفية البيانات**:
   - السابق: جلب جميع مشاكل الجلد دون تصفية
   - الجديد: جلب مشاكل الجلد المفلترة حسب `tool_title`

3. **إضافة `category` إلى dependencies**:
   - الآن يعاد جلب البيانات تلقائياً عند تغيير الفئة

## آلية العمل

### عند إضافة منتج في المخزن:

#### السيناريو 1: منتج من فئة الشعر (hair_care)
```
اختيار الفئة: hair_care
  ↓
getCategoryToolTitle('hair_care') → 'العناية بالشعر'
  ↓
Query: SELECT * FROM skin_problems WHERE tool_title = 'العناية بالشعر'
  ↓
النتيجة (مشاكل الشعر فقط):
  ✓ قشرة الرأس (Dandruff)
  ✓ جفاف الشعر (Dry Hair)
  ✓ تساقط الشعر (Hair Loss)
  ✓ الشعر الدهني (Oily Hair)
  ✓ تقصف الشعر (Split Ends)
```

#### السيناريو 2: منتج من فئة البشرة (skincare)
```
اختيار الفئة: skincare
  ↓
getCategoryToolTitle('skincare') → 'العناية بالبشرة'
  ↓
Query: SELECT * FROM skin_problems WHERE tool_title = 'العناية بالبشرة'
  ↓
النتيجة (مشاكل البشرة فقط):
  ✓ حب الشباب (Acne)
  ✓ البقع الداكنة (Dark Spots)
  ✓ جفاف البشرة (Dryness)
  ✓ البشرة الدهنية (Oily Skin)
  ✓ البشرة الحساسة (Sensitivity)
  ✓ التجاعيد (Wrinkles)
```

## الفوائد

✅ **فصل كامل** - لا اختلاط بين مشاكل الشعر والبشرة  
✅ **تحديث ديناميكي** - البيانات تتغير تلقائياً عند تغيير الفئة  
✅ **تجربة أفضل** - المستخدم يرى فقط المشاكل ذات الصلة  
✅ **منع الأخطاء** - لا يمكن ربط مشكلة شعر بمنتج للبشرة والعكس  

## الاختبار

### خطوات التحقق:

1. **اذهب إلى صفحة المخزن** (`/inventory`)

2. **اختبر فئة الشعر**:
   - انقر على "إضافة منتج يدوي"
   - اختر الفئة: "العناية بالشعر"
   - تحقق: يجب أن تظهر **فقط** مشاكل الشعر
   - ✓ قشرة الرأس، جفاف الشعر، تساقط، إلخ

3. **اختبر فئة البشرة**:
   - غيّر الفئة إلى: "العناية بالبشرة"
   - تحقق: يجب أن تظهر **فقط** مشاكل البشرة
   - ✓ حب الشباب، بقع داكنة، جفاف بشرة، إلخ

4. **تأكد من التحديث الديناميكي**:
   - غيّر الفئة من الشعر للبشرة
   - قائمة المشاكل يجب أن تتحدث فوراً

## ملاحظات

- المقابلة تدعم جميع متغيرات أسماء الفئات
- إذا لم تطابق الفئة أي من الخيارات، لن تظهر أي مشاكل
- التغيير لا يؤثر على الصفحات الأخرى (ProductCard، RecommendationPage، إلخ)

## التحقق من قاعدة البيانات

تأكد من أن جدول `skin_problems` يحتوي على:
```sql
-- Haircare problems
INSERT INTO skin_problems (..., tool_title) VALUES (..., 'العناية بالشعر');

-- Skincare problems  
INSERT INTO skin_problems (..., tool_title) VALUES (..., 'العناية بالبشرة');
```

البيانات التي أدرجتها في الطلب الأول موجودة بالفعل وتحتوي على `tool_title` الصحيح.
