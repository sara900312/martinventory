## الخطوات المطلوبة من المستخدم

### ✅ الخطوة 1: جعل الدلو عام
```
Supabase Dashboard → Storage → popup-media 
→ (⋮) Edit → Enable "Public bucket" → Save
```

### ✅ الخطوة 2: التحقق من النجاح
- الصور والفيديوهات ستظهر تلقائياً
- رسالة التشخيص ستختفي من لوحة التحكم

### ✅ الخطوة 3: رفع وسائط جديدة
- استخدام نفس واجهة رفع الملفات كما هو
- الملفات ستُحفظ في الدلو العام

## معالجة الأخطاء المحسّنة

### في الكود:
```javascript
// رسائل خطأ واضحة
console.log('Image loaded successfully:', popup.image_url);
console.error('Failed to load image:', popup.image_url, e);
```

### في المتصفح (F12 → Console):
```
✅ Image loaded successfully: https://...
❌ Failed to load image: https://... 
   Error: CORS error or bucket not public
```

### في لوحة التحكم:
- رسالة تحذير حمراء إذا كان الدلو غير عام
- تعليمات خطوة بخطوة لحل المشكلة

---

## الحالات الثلاث

### 1️⃣ الدلو عام ✅
```
- الصور والفيديوهات تظهر بشكل طبيعي
- لا توجد رسالة تحذير
- يعمل كل شيء بشكل صحيح
```

### 2️⃣ الدلو غير عام ⚠️
```
- الصور والفيديوهات لا تظهر (خلفية بيضاء)
- رسالة تحذير في لوحة التحكم
- خطأ CORS في جهاز المطورين (F12)
```

### 3️⃣ الدلو غير موجود ❌
```
- الإعلان لا يحتوي على صور/فيديوهات
- رسالة خطأ في الكود
- يجب إنشاء الدلو أولاً
```

## الدعم الإضافي

إذا استمرت المشكلة:
1. افتح جهاز المطورين (F12)
2. انقر على "Network"
3. حاول تحميل صورة
4. ابحث عن الطلب الذي فشل
5. اقرأ رمز الخطأ:
   - **403**: الدلو ليس عام
   - **404**: الملف غير موجود
   - **CORS Error**: مشكلة في الرؤوس
