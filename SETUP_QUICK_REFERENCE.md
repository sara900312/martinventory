# 🚀 دليل سريع - نظام التوصيات
## Quick Reference Guide - Recommendations System

---

## 📍 الخطوات الأساسية (5 دقائق)
### Basic Steps (5 minutes)

### 1️⃣ اذهب إلى المخزن
```
Dashboard → المخزن
أو مباشرة: https://your-domain/inventory
```

### 2️⃣ أضف منتج جديد
```
اضغط: "+ إضافة منتج جديد"
```

### 3️⃣ ملأ البيانات الأساسية
```
✓ الاسم (عربي وإنجليزي)
✓ السعر
✓ الكمية
✓ الوصف
✓ الصورة الرئيسية
```

### 4️⃣ اختر الفئة المهمة!
```
⭐ اختر: skincare أو hair_care
   (فقط هذه الفئات تدعم التوصيات)
```

### 5️⃣ قسم التوصيات
```
في "🧴 Skin & Routine Settings"

الخطوة A: اختر مشاكل البشرة
  ☑ acne (حب الشباب)
  ☑ dryness (جفاف)
  ☑ oiliness (دهنية)
  (اختر جميع المشاكل التي يعالجها المنتج)

الخطوة B: اختر نوع الروتين
  • 🌅 Morning (صباحي)
  • 🌙 Night (ليلي)
  • ✨ Special (خاص)
```

### 6️⃣ الوصف القصير (مهم!)
```
بالعربية:
"وصف قصير للمنتج (50-100 حرف)"

بالإنجليزية:
"Short product description (50-100 chars)"
```

### 7️⃣ انشر المنتج
```
☑ Published = YES

اضغط: "حفظ" أو "إضافة المنتج"
```

## 🎨 أنواع الروتين شرح سريع
### Routine Types

### 🌅 صباحي (Morning)
```
خفيف • سريع الامتصاص • لا يترك دهون

أمثلة المنتجات:
• واقي الشمس
• تونر منقي
• مرطب خفيف
• مصل فيتامين سي
```

### 🌙 ليلي (Night)
```
ثقيل • تغذية عميقة • علاجي

أمثلة المنتجات:
• مصل مكثف
• ماسك علاجي
• كريم غني
• زيت طبيعي
```

### ✨ خاص (Special)
```
قوي • متخصص • علاجي مكثف

أمثلة المنتجات:
• مقشر قوي
• مصل علاجي
• ماسك متقدم
• جل مسح
```

## 🆘 مشاكل شائعة وحلولها
### Common Issues & Solutions

### ❌ لا تظهر خيارات التوصيات
```
✓ تأكد الفئة = skincare أو hair_care
✓ أعد تحميل الصفحة
✓ تأكد قاعدة البيانات تحتوي skin_problems
```

### ❌ المنتج لا يظهر في التوصيات
```
✓ تأكد Published = true
✓ تأكد Stock > 0
✓ تأكد اخترت مشكلة + روتين
✓ انتظر دقيقة واحدة (cache)
✓ أعد تحميل صفحة التوصيات
```

### ❌ الروتين لم يظهر
```
✓ الفئة يجب = skincare أو hair_care
✓ اختر مشكلة بشرة أولاً ثم سيظهر الروتين
```
