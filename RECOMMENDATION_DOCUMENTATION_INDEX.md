## 📞 الدعم والمساعدة

### إذا واجهت مشكلة:

1. **جرب الاختبار السريع:**
   ```bash
   # في SQL Editor
   SELECT * FROM products 
   WHERE skin_problems ILIKE '%حب الشباب%' 
   LIMIT 5;
   ```

2. **اقرأ قسم "استكشاف الأخطاء":**
   - في RECOMMENDATION_SETUP_GUIDE.md

3. **تحقق من Console:**
   - في متصفح الويب (F12 → Console)
