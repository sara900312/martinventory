### جدول maintenance
```sql
CREATE TABLE maintenance (
  id uuid PRIMARY KEY,
  section_name text,
  is_active boolean,
  start_time timestamp,
  end_time timestamp,
  reason text,
  created_at timestamp,
  updated_at timestamp
)
```

## 📝 ملاحظات مهمة

### تعديل سياسة الوصول

في ملف SQL، قد تحتاج إلى تعديل:

```sql
CREATE POLICY "Allow admin full access on maintenance"
ON public.maintenance
FOR ALL
USING (
  -- استبدل هذا برقم المستخدم أو الدور الفعلي
  auth.uid() IN (SELECT id FROM auth.users WHERE raw_user_meta_data->>'role' = 'admin')
);
```

### إضافة أقسام جديدة

1. حدّث ملف SQL (الـ constraint)
2. حدّث `SECTIONS` في `MaintenanceManagementPage.jsx`
