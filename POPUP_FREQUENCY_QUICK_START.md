# 🚀 البدء السريع - نظام تكرار الإعلانات

## في 3 خطوات فقط:

### ✅ الخطوة 1: تشغيل الترقية (مرة واحدة)
```sql
-- افتح Supabase Dashboard → SQL Editor وشغّل:
ALTER TABLE public.popup_hero 
ADD COLUMN IF NOT EXISTS frequency_interval INTEGER DEFAULT 5;
```

### ✅ الخطوة 2: إنشاء إعلان
1. اذهب إلى **إدارة الإعلانات**
2. اختر **إنشاء إعلان**
3. ملّ البيانات الأساسية (العنوان، الوصف، الصورة)

### ✅ الخطوة 3: اختيار التكرار
في قسم "تكرار العرض"، اختر أحد الخيارات:

| الخيار | الشرح | مثال |
|--------|-------|------|
| 🔄 **عرض في كل مرة** | يظهر دائماً | عروض عاجلة |
| 👁️ **عرض مرة واحدة** | يظهر مرة فقط | ترحيب بالزائرين |
| ⏱️ **عرض كل X مرات** | كل N زيارة | تذكيرات منتظمة |

---

## 📝 أمثلة التكوين

### مثال 1: إعلان يومي عاجل
```
الخيار: عرض في كل مرة
النتيجة: يظهر في كل زيارة خلال اليوم
```

### مثال 2: ترحيب بالزائرين الجدد
```
الخيار: عرض مرة واحدة فقط
النتيجة: يظهر مرة واحدة، ثم لا يظهر إلا بعد مسح localStorage
```

### مثال 3: تذكير كل أسبوع
```
الخيار: عرض كل X مرات
القيمة: 20 (تقريباً زيارة واحدة يومياً لمدة 3 أسابيع)
النتيجة: يظهر الإعلان كل 3 أسابيع
```

---

## 🧪 التحقق من أن كل شيء يعمل

### في متصفح العميل:
```javascript
// افتح Browser Console (F12) وانسخ:

// 1. تحقق من البيانات المحفوظة:
localStorage

// 2. شاهد جميع بيانات الإعلانات:
Object.keys(localStorage).filter(k => k.startsWith('popup_frequency_'))

// 3. اطبع بيانات إعلان محدد:
JSON.parse(localStorage.getItem('popup_frequency_<popup-id>'))

// 4. امسح جميع الإعلانات (لإعادة الاختبار):
Object.keys(localStorage)
  .filter(k => k.startsWith('popup_frequency_'))
  .forEach(k => localStorage.removeItem(k))
```

## 🎯 حالات الاستخدام الشائعة

| الحالة | الخيار | الإعدادات |
|--------|--------|----------|
| **عرض خصم محدود الوقت** | عرض في كل مرة | لا توجد |
| **ترحيب بزائر جديد** | عرض مرة واحدة | لا توجد |
| **استطلاع رأي** | عرض كل 15 زيارة | interval: 15 |
| **تنبيه مهم** | عرض في كل مرة | لا توجد |
| **عرض موسمي** | عرض كل 10 زيارات | interval: 10 |

---

## ⚡ نصائح مهمة

1. **لا تنسَ التفعيل**: لا يظهر الإعلان ما لم تضغط "نشط"
2. **الجدولة مهمة**: تأكد من `start_date` و `end_date`
3. **اختبر دائماً**: استخدم وضع المعاينة قبل النشر
4. **امسح localStorage للاختبار**: عند تغيير الإعدادات
5. **راقب Console**: للتأكد من عمل النظام

---

## 🔧 استكشاف المشاكل

| المشكلة | الحل |
|--------|------|
| الإعلان لا يظهر | تحقق: `is_active=true`, `start_date < now`, `end_date > now` |
| يظهر دائماً (عندما اخترت مرة واحدة) | امسح localStorage وأعد التحميل |
| التكرار خاطئ | تحقق من `frequency_interval` في قاعدة البيانات |

## 📊 معادلة حساب التكرار

```
الزيارة الحالية = X
آخر عرض كان في = الزيارة Y
عدد الزيارات المطلوبة = N

يظهر الإعلان عندما: (X - Y) ≥ N
```

**مثال:**
- الإعدادات: عرض كل 5 زيارات
- الزيارة 1: يظهر ✅ (أول عرض)
- الزيارة 2-5: لا يظهر
- الزيارة 6: يظهر ✅ (مرت 5 زيارات)
- الزيارة 7-10: لا يظهر
- الزيارة 11: يظهر ✅ (مرت 5 زيارات أخرى)
